event parameter

for any event handler .click(), .mouseover(), etc. data about the event is passed to the function

It may contain...

the coordinates of the cursor on the page

which key was pressed

where or whether the mouse was clicked

the event parameter also exposes some functions

.preventDefault();

prevents the normal action of an event from occurring

example: disabling an anchor tag from navigating to its href

.isDefaultPrevented();

returns a boolean value {true or false} based on whether .preventDefault(); was used on that event
